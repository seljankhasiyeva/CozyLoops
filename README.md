🧶 CozyLoops - Handmade E-Commerce Platform
CozyLoops is a full-stack e-commerce application designed for selling high-quality, handmade knitted products. The platform provides a seamless shopping experience with a focus on artisanal craftsmanship.

🚀 Features
- User Authentication: Secure Login and Registration using ASP.NET Core Identity and JWT Tokens.

- Product Management: Dynamic product listing with "New Arrival" tags and detailed product pages.

- Shopping Cart (Bucket): Full cart functionality (Add, update quantity, remove) linked to user accounts.

- Checkout System: Order processing with address management.

- Settings Management: Dynamic website settings managed via the database.

- Responsive Design: Fully optimized for both desktop and mobile devices.

🛠️ Tech Stack
Backend
- Framework: ASP.NET Core 8.0 / 9.0 Web API

- Database: MS SQL Server (via Entity Framework Core)

- Security: JWT (JSON Web Tokens) & Identity Framework

- Architecture: Repository Pattern

Frontend
- Technologies: HTML5, CSS3 (Modern UI/UX), JavaScript (ES6+)

- State Management: LocalStorage for Auth persistence

- API Communication: Fetch API
